About Me


Richie Yin


A passionate software engineer with a strong background in leadership and a lifelong love for technology. Having been in leadership roles since grade school, I've honed my management skills through diverse experiences such as leading group projects, running a restaurant, overseeing a gym sales team, and managing a business.

Growing up in an immigrant family with no prior computer experience, I became the go-to IT person for my family at just 8 years old. My curiosity for coding began at the age of 12, but it wasn't until recently that I fully immersed myself in the world of software development.

Over the past 14 weeks, I committed myself to an intensive coding bootcamp, where I gained proficiency in a wide range of programming languages and frameworks. My skills include:


  • Web Fundamentals: HTML5 / CSS3 / JavaScript / jQuery
  • Python: Python / Flask / Flask + MySQL
  • MERN: JavaScript / MongoDB / Express / React / Node.js / Deployment
  • Java: Java / Spring / Java + MySQL
  • Databases: MongoDB / MySQL
  • Extras: APIs / RESTful Routing / Git / Github
